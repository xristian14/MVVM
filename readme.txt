Приложение в конструкторе Model создает в новом потоке бесконечный цикл который с задержкой в 1 сек увеличивает свойство Clicks на 1.
При обновлении этого свойства, обновляется такое же свойство в ViewModel, и дальше обновляется значение в View.
Кнопка на View через команду в ViewModel добавляет в ObservableCollection<DataSource> _dataSources модели текстовое значение Clicks.
После добавления значения в _dataSources Model, обновляется такое же свойство во ViewModel, после чего обновляется ListView во View.
